Kolibri is an elegant static site generator built in Python. It doesn't lock you in any framework way, instead it lets you have it your way. A blazing fast, declarative, expressive, zero dependency, no virtual dom, view library that lets you write Javascript Template Literals in your HTML.
Efficiently it re-renders parts that should be rendered on the DOM.
Also provides a simple state management
At the end, it gives you back your HTML. No transpilation, compilation
